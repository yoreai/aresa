## Overview

ARESA Studio is a unified command-line interface and web UI for querying 8+ database types with a single tool. It eliminates the need to switch between different database clients by providing a consistent experience across all major database platforms.

---

## Supported Databases

- **PostgreSQL** - Full SQL support with schema exploration
- **MySQL** - Complete query and schema management
- **SQLite** - Embedded database queries
- **ClickHouse** - Columnar analytics queries
- **BigQuery** - Google Cloud data warehouse
- **DuckDB** - In-process analytical database
- **Snowflake** - Cloud data platform
- **Databricks** - Unified analytics platform

## Key Features

### Universal Query Interface
Execute SQL queries against any supported database using the same familiar interface. No need to learn different tools or syntaxes.

### Schema Exploration
Browse tables, columns, and relationships across all your data sources. Understand your data structure at a glance.

### Connection Management
Hot-reloading configuration allows you to add, modify, or remove database connections without restarting the application.

### Query History
Track your query history across sessions. Easily recall and re-run previous queries.
